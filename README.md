# Crime Management System (DBMS Project)

This project is based on Database Management System (DBMS). 
The Crime Management System helps to store and manage crime records efficiently.

Features:
- Store criminal records
- Manage FIR details
- Track case status
- Maintain police records
- Easy data retrieval using SQL queries

Technologies Used:
- DBMS
- SQL
- ER Diagram
- Normalization

Purpose:
The system improves data organization and helps authorities manage crime-related information efficiently.
